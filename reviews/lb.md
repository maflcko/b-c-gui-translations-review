# Translations Review by LLM (✨ experimental)

The review quality depends on the LLM and the language. Currently, a fast LLM without rate limits is used. If you are interested in better quality for a specific language, please file an issue to ask for it to be re-run with a stronger model.

```
        <source>Export the data in the current tab to a file</source>
        <translation>Exportéiert déi Dateien op der aktueller Tabell an eng Datei.</translation>
    
YES: The word "Aktueller" is misspelled. It should be "aktueller". Additionally, the word "Tabell" implies a table, but the original text refers to a "tab". A better translation would be "Tab".

Exportéiert d'Dateien op der aktueller Tab an eng Datei.
```
