# Translations Review by LLM (✨ experimental)

The review quality depends on the LLM and the language. Currently, a fast LLM without rate limits is used. If you are interested in better quality for a specific language, please file an issue to ask for it to be re-run with a stronger model.

```
        <source>Right-click to edit address or label</source>
        <translation>Мекенжай немесе белгі өңдеу үшін оң клик</translation>
    
YES  
The translation uses Cyrillic script ("Мекенжай немесе белгі өңдеу үшін оң клик"), but the requested language is 'kk@latin', which requires the Kazakh language in Latin script. Furthermore, "оң клик" is a direct transliteration of "right click" using a Russian-influenced term "клик" (click), which might be better rendered in Kazakh as "оң жақ батырмамен басыңыз" or simply "оң батырманы басыңыз" for clarity.

Correct translation in Kazakh Latin script:  
"Oñ batırmanı basıp, mekenday nemese belgini oñdäw"  
(Meaning: "Right-click to edit address or label")

Alternatively, a smoother version could be:  
"Oñ batırmanı basıp mekenday nemese belgini oñdäñiz"
```
