# Translations Review by LLM (✨ experimental)

The review quality depends on the LLM and the language. Currently, a fast LLM without rate limits is used. If you are interested in better quality for a specific language, please file an issue to ask for it to be re-run with a stronger model.

```
        <source>Create a new address</source>
        <translation>လိပ်စာအသစ်ယူမယ်။</translation>
    
YES. The translation is not accurate and does not convey the intended meaning. The phrase "လိပ်စာအသစ်ယူမယ်" in Burmese translates to "I will take a new address" or "I want a new address". This is an imperative or a statement of intent, rather than an instruction or a request to create a new address.

The correct translation should be an imperative form, similar to "Create a new address."

Correct translation: **လိပ်စာအသစ်တစ်ခု ဖန်တီးပါ။**
```

```
        <source>Enter address or label to search</source>
        <translation>လိပ်စာရိုက်ထည့်ပါ</translation>
    
YES. The translation is missing the phrase "or label to search".
Correct translation: လိပ်စာသို့မဟုတ် အမည်ပြောင်းကို ရှာဖွေရန် ထည့်သွင်းပါ
```

```
        <source>Choose the address to send coins to</source>
        <translation>လိပ်စာကိုပေးပို့ဖို့လိပ်စာရွေးချယ်ပါ</translation>
    
YES: The translation is grammatically incorrect and awkward in Burmese. It literally translates to "Choose the address to send to the address."
The correct translation should be: "လိပ်စာကိုရွေးချယ်ပါ" (Choose the address) or "ဒင်္ဂါးများပေးပို့ရန်လိပ်စာကိုရွေးချယ်ပါ" (Choose the address to send coins).
```
