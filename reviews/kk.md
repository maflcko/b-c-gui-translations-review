# Translations Review by LLM (✨ experimental)

The review quality depends on the LLM and the language. Currently, a fast LLM without rate limits is used. If you are interested in better quality for a specific language, please file an issue to ask for it to be re-run with a stronger model.



<details><summary>kk</summary>

[If the result is of low quality, please file an issue to find a better LLM for this language.](../../issues/new?title=%5Bkk%5D%20low%20quality)


```
        <source>Right-click to edit address or label</source>
        <translation>Мекенжай немесе белгі өңдеу үшін оң клик</translation>
    
YES. The translation is grammatically incorrect and does not convey the intended meaning. The verb "edit" should be in a more appropriate form for the imperative command.

Correct translation: Мекенжайды немесе белгіні өңдеу үшін тінтіштің оң жақ түймесін басыңыз.
```

```
        <source>Copy the currently selected address to the system clipboard</source>
        <translation>Таңдалған мекенжайды жүйенің айырбастау буферіне көшіру</translation>
    
YES - The translation incorrectly uses "айырбастау буфері" (exchange buffer) which is not the standard term for "clipboard" in Kazakh. The correct term is "жетім" or "алмасу буфері".

NO
```

```
        <source>%1 and %2</source>
        <translation>%1 немесе %2</translation>
    
YES, The translation uses "немесе" (or) instead of "and". The correct translation should be "%1 және %2".
```

```
        <source>Sign messages with your Bitcoin addresses to prove you own them</source>
        <translation>Хатқа Bitcoin мекенжайларын қосып, олар сізге тиесілі екенін дәлелдеу</translation>
    
YES - The translation is inaccurate and misses the core meaning of the English text. The provided translation means "Add Bitcoin addresses to a letter and prove they belong to you", which is not the intended meaning of signing messages to prove ownership.
Correct translation: Bitcoin мекенжайларымен хабарламаларға қол қойып, олардың сізге тиесілі екенін дәлелдеңіз
```

```

        <translation>Хат тексеріп, берілген Bitcoin мекенжайлары қосылғанына көз жеткізу</translation>

Correct translation: "Хабарларды көрсетілген Bitcoin мекенжайларымен қол қойылғанына көз жеткізу үшін тексеріңіз"
```
